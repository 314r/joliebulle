# vuepack

To start:

```bash
$ npm install
```

To develop:

```bash
$ npm run dev
# in a new tab, run electron app
$ npm run app
```

To build for production:

```bash
$ npm run build
```

To lint you code:

```bash
$ npm run lint
```
